자주 묻는 질문

AI 글 생성 기능은 구글 제미나이(Gemini) AI를 기반으로 작동하며, 원활한 사용을 위해 사용자 본인의 API 키를 등록하여 사용해야 합니다. API 사용에 따른 비용은 구글 정책에 따르며, 사용자 계정별로 별도 발생합니다.

네, 키워드만 입력하시면 AI가 콘텐츠 생성, 이미지 제작, 블로그 발행까지 모든 과정을 자동으로 처리합니다. 단, 초기 설정과 전략 수립은 사용자의 몫입니다.
기술 지원은 프로그램 작동 오류 및 설치 관련 사항에 한하며, 블로그 운영 전략·마케팅 성과(조회수, 노출 등)에 대한 컨설팅은 포함되지 않습니다.
현재 네이버 블로그만을 지원하며, 향후 워드프레스, 티스토리, 브런치 등 다양한 플랫폼으로 확대 예정입니다.
AI가 생성하는 모든 콘텐츠와 이미지는 고유하게 제작되며, 상업적 라이센스가 있는 소스만 사용합니다. 저작권 걱정 없이 사용하실 수 있습니다.
